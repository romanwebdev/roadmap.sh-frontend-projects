# Pomodoro Timer

A simple Pomodoro Timer application built with Vue, HTML, CSS, and JavaScript. The app allows users to start, stop, and resume a customizable Pomodoro timer. It automatically switches between work sessions and break sessions, and tracks completed work intervals. It is optimized for accessibility and works seamlessly on both desktop and mobile devices.

## Prerequisites

Install [Node.js](https://nodejs.org/)

To check if Node.js is installed, run:

```sh
node -v
```

## Usage

To run the application locally:

1. **Clone or download the repository and navigate to the project directory:**

```sh
cd roadmap.sh-frontend-projects/projects/21-pomodoro-timer
```

2. **Install dependencies:**

```sh
npm install
```

3. **Start the development server:**

```sh
npm run dev
```

4. **Open your browser and visit:**

```
http://localhost:5173
```

## Technologies

HTML, CSS, JavaScript, Vue, npm, Vite

## Requirements

- [x] User should be able to start stop and resume a pomodoro timer
- [x] User should be able to configure the default interval configuration; default work session should be 25 minutes, short break should be 5 minutes and longer break after 4 work sessions should be 15 minutes
- [x] Application should display the current session type (e.g., Work, Short Break, Long Break)
- [x] It should also track the number of tracked work sessions
- [x] Play a sound when a session ends to notify the user
- [x] Ensure the app is accessible and visually appealing on both desktop and mobile devices
